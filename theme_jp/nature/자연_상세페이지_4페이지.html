#自然
大関嶺治癒の森
韓国山林福祉院が運営する７つの治癒の森のうちの１つで、鬱蒼とした金剛松が壮観です。 様々な難易度の8コースがあり、一番長いコースの長さも1.6km程度で気軽に歩いてみることができます。 金剛松展望台から大関嶺を一望でき、澄んだ空気とともに心身ともに休息をとることができます。 事前に予約すれば多様な山林治癒プログラムも体験できますが、妊婦のための「シン・サイムダン森胎教」、 青少年のための「スリスリ森学校」、一般人を対象にした「ソルソオルテラピー」、家族単位で体験する「ソルスプルトクトクファミリー」、疎外階層障碍のある方々や社会的に困難な立場の方々にエネルギーを充電してくれる「ソルヒャン分かち合いの森」などの多様なオーダーメード型プログラムがあります。
ホームページ : https://www.fowi.or.kr/user/contents/contentsView.do?cntntsId=140
住所 : 江原道江陵市大関嶺旧道127-42
電話番号 : 033-642-8651

大邱樹木園
大邱樹木園はかつてのゴミ埋立地に建てられた樹木園で、環境問題を解決するために建てられました。 針葉樹園、広葉樹園、花木園、野生草花園、薬用植物園、染料園など21ヶ所の多様な園で構成されており、多様な盆栽や樹石も展示されています。 全25のテーマ庭園で構成されており、都心から近い都市型樹木園で、大邱を観光しながら短時間の訪問にも良い場所です。 特に樹木園近くの山と頭流公園を一緒に見て回るのに最適です。
ホームページ : http://www.daegu.go.kr/Forestry
住所 : 大邱広域市達西区花岩路342
電話番号 : 053-803-7270

大明流水池&達城湿地
大明遊水地は洛東江の氾濫を防ぐために1995年に作られた人工貯水池で、サッカー場30カ所規模(26万㎡)の湿地にはススキがいっぱいで、 秋には数千万のススキの花が風に舞う姿が壮観で写真がきれいに撮られる所として有名な場所です。達城湿地は洛東江と琴湖江が出会うトゥムルモリに積もった堆積物が形成された湿地で、 200万㎡に達する面積を誇る大型湿地です。 絶滅危惧種であるチョウゲンボウ、希少植物カワニナなど多様な生物が生息しています。 過去には湿地の出入りが禁止されていましたが、生態復元事業を完了し、最近開放されました。 湿地にはイチョウ、カエデ、桜、ケヤキが作り出した林道が広がり、アンタクトヒーリングが体験できます。
ホームページ : https://dalseo.daegu.kr/tour/index.do
住所 : 大邱広域市達西区大川洞816
電話番号 : 053-667-2171

摩尼山治癒の森
摩尼山は江華島で最も高い山です。 摩尼山の「癒しの森」はアンタクト観光地に指定されており、1kmほどの長くないコースなので老若男女誰でも簡単に見回ることができます。 松や杉が生い茂る林道を歩きながらさわやかな空気を吸い、憩いの場でしばらく座って瞑想をするとストレスは吹き飛び、癒される気分になれます。
ホームページ : http://www.ganghwafocus.com/exp/index.php
住所 : 仁川広域市江華郡華道面摩尼山路675番キル18
電話番号 : 070-4101-8308
