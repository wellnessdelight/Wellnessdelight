#韓方
ナタン薬膳料理
ナタン薬膳料理は、薬となる韓国料理の薬膳料理とウェルビーイングティー、そして化学調味料を排除し、材料本来の味と色を強調する健康料理をご提供します。 薬剤効能と栄養成分を計算して味と形、治療効果まで得られる機能食品でメニューを構成して食べることができます。
ホームページ : -
住所 : 釜山鎮区ソンジゴク路51番キル4
電話番号 : 051-504-7733
#ビューティー/スパ
スパ1899海雲台店
スパ1899は人の体を治めるテラピストの熟練した手つきで紅参が持つ価値を内面と外面の美しさに昇華し、真心を込めたパフォーマンスとセラピーを施すスパです。 1899年から続いてきた職人精神で独自開発した「紅参エネルギーサーキュレーションセラピー技法」は、時間が経つにつれて光を放つ紅参の凝縮された生命力を感じられるよう、体の円滑な循環を助けます。
ホームページ : http://www.spa1899.co.kr/sp1899/
住所 : 釜山広域市海雲台区海雲台海辺路277タオムビル6階
電話番号 : 051-747-8036
#ヒーリング/瞑想
大韓仏教曹渓宗梵魚寺テンプルステイ
梵魚寺テンプルステイは、静かな山寺で真の幸せな旅行を訪れる人々に感じていただける場所で、自分を振り返ってリフレッシュできる時間を提供します。 林道瞑想ウォーキング、早朝礼拝、108拝書院酒づくり、早朝瞑想による参加者の内面の安定と癒しを得ることができます。 韓国/外国人を対象に体験型「禁語テンプルステイ」と休息型「憩いテンプルステイ」プログラムを常設運営しています。
ホームページ : http://www.beomeo.kr/templestay/ts01.php
住所 : 釜山広域市金井区梵魚寺で250梵魚寺
電話番号 : 051-508-5726
#自然
国立大雲山治癒の森
国立大雲山治癒の森は清涼な渓谷とアベマキ、ヒノキが調和した林道で様々な山林治癒プログラムを運営しています。 山林治癒プログラムは森の多様な環境要素を活用して人体の免疫力を高め身体と精神の健康を回復させる活動で、 「森の中の五感遊び場」から「森トックス」まで老若男女皆が楽しめるプログラムが10種類で構成されています。
ホームページ : https://www.fowi.or.kr/user/contents/contentsView.do?cntntsId=161
住所 : 蔚山広域市蔚州郡温陽邑大雲上大吉225-92
電話番号 : 052-255-9800
